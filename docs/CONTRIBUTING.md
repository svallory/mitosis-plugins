# Contributing to the Starlight Template

First off, thank you for considering contributing to this template!

## How Can I Contribute?

### Reporting Bugs

- Use the GitHub issue tracker to report bugs.
- Provide a clear description and steps to reproduce.

### Suggesting Enhancements

- Enhancements are welcome! Please open an issue first to discuss.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.

## Stylecards

- Try to follow the existing code style.
- Be respectful and professional.
